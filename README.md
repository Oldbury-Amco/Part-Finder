# AMCO Part Finder

Static GitHub Pages app for mobile-friendly AMCO part lookup.

## Files
- `index.html`
- `style.css`
- `app.js`
- `parts_data.json`
- `amco-logo.webp`
- `.nojekyll` (recommended in repo root)

## Update on GitHub
Replace the files above in the root of the repository, commit, and allow GitHub Pages a minute to redeploy.
